# GitHub Project Setup Skill

## Purpose

This skill initializes and structures professional GitHub repositories and project management workflows for data engineering and cloud engineering projects.

The skill should automate and standardize GitHub project setup activities to reduce manual project management overhead and ensure consistency across repositories.

---

## Responsibilities

The skill should be capable of:

* Creating standardized GitHub project structures
* Generating repository initialization files
* Suggesting professional repository naming conventions
* Creating project boards
* Generating Epics and Issues
* Creating Sub-Issues and implementation tasks
* Suggesting Labels and Priorities
* Creating Milestones and roadmap phases
* Generating issue templates and pull request templates
* Structuring work into implementation phases

---

## Expected GitHub Structure

The skill should generate project structures such as:

* Epics
* Features
* Tasks
* Subtasks
* Milestones
* Labels
* Priorities
* Project columns

## Preferred Workflow

The skill should:

* prioritize practical implementation
* organize work into logical engineering phases
* avoid unnecessary project management complexity
* structure tasks clearly and concisely and in bullet points
* generate scalable project boards for data engineering projects
